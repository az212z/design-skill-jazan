# ديزاين سكِل — استوديو تصميم داخلي في جازان

موقع تعريفي عربي (RTL) لاستوديو **Design Skill / ديزاين سكِل** للتصميم الداخلي في مدينة جازان.

## نبذة
- **الاسم:** ديزاين سكِل (Design Skill)
- **التصنيف:** استوديو تصميم داخلي (InteriorDesigner)
- **المدينة:** جازان، المملكة العربية السعودية
- **التقييم:** 4.5 على خرائط قوقل (149 تقييم) — مذكور كما هو، بلا مبالغة.

## لماذا اخترنا هذا النشاط
نشاط تصميم داخلي بتقييم عالٍ (4.5) وحضور حقيقي في جازان، وجمهوره يبحث عن جودة بصرية — وهي بالضبط القيمة التي يبيعها الموقع.

## فكرة التصميم
"Modern Architectural Minimal": لوحة فحمي + رملي/جريج + أوف-وايت دافئ + لمسة طينية (terracotta) واحدة. خطوط هندسية نظيفة، مساحات تتنفّس، خطّ عرض ثقيل (Tajawal) مع جسم نظيف (IBM Plex Sans Arabic). معرض الأعمال مبنيّ بعناصر CSS معمارية بدلًا من صور المنافسين.

## التقنية
- HTML/CSS/JS فانيلا بلا أي خطوة بناء (no build step) — جاهز لـ GitHub Pages.
- لا مكتبات ثقيلة. خطوط Google عبر preconnect + display=swap.
- `IntersectionObserver` لكشف السكرول مع fallback يضمن ظهور كل المحتوى.

## حالة الـ Backend
**غير مربوط بأي خادم.** نموذج طلب التصميم تجريبي: يتحقّق من المدخلات، ثم يبني رسالة **واتساب** مهيّأة مسبقًا (`wa.me/966553368805`)، ويحفظ نسخة محلية في `localStorage` فقط، مع إشعار (toast) نجاح.

## مصدر الصور
صورة واحدة فقط مُستخدمة: واجهة الاستوديو نفسه (`assets/img/ds-17343.jpg`) من خرائط قوقل، بعد تنقية (Curation). استُبعدت صور أربع لأنها لافتات منافسين، أو render بعلامة مائية لجهة أخرى، أو واجهة نشاط مختلف بلوحة سيارة ظاهرة. بقية الأقسام البصرية مبنية بـ CSS.
